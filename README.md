DiaCare AI is an AI-powered diabetes risk prediction web application built using **Flask**, **Machine Learning**, and **Google Gemini AI**. The application predicts diabetes risk based on **Age, Gender, BMI, and Blood Glucose Level** using a Logistic Regression model. It also provides confidence scores, health insights, personalized recommendations, downloadable PDF health reports, and an AI Health Assistant for answering health-related queries.

---

## ✨ Features

- 🔐 Secure User Registration & Login
- 🤖 Diabetes Risk Prediction using Logistic Regression
- 📊 Interactive Dashboard with Health Metrics & Charts
- ❤️ BMI, Blood Glucose & Health Score Analysis
- 📜 Prediction History
- 📄 Downloadable PDF Health Report
- 💬 AI Health Assistant powered by Google Gemini
- 💾 Persistent Chat History
- ⭐ Feedback & Rating System
- 📱 Responsive and User-Friendly Interface

## 🚀 Installation

```bash
git clone https://github.com/YOUR_USERNAME/DiaCare-AI.git
cd DiaCare-AI
pip install -r requirements.txt
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
